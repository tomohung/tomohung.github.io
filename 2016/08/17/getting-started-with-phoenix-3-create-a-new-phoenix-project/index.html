Getting started with Phoenix - 3. Create a new Phoenix project

Aug 17, 2016

Create a new Phoenix project

Install Phoenix

可以參考Phoenix官網安裝方法
另外我們要使用MySQL，所以也要記得安裝，網路上教學文章很多，這裡就跳過

Create the Blog application

打開終端機執行以下程式

--database mysql是為了將預設資料庫PostgreSQL指定成MySQL。
而blog是我們專案的名稱

接下來會提示是否安裝相依套件
Fetch and Install dependencies? [Yn]

選擇Y之後，程式應該會自動執行這兩行

如果npm install沒安裝的話，前端套件會沒有，你看到的就是一個醜醜的網站，因為Phoenix已經預設幫你用了Bootstrap的css檔案，和一點點客製化的內容。

想要有完整的Bootstrap功能，包含Javascript，可以參考我另一篇文章Phoenix with Bootstrap

接下來進入專案並且執行server

Server運行起來後，打開瀏覽器輸入localhost:4000，你應該可以看到Phoenix的觀迎首頁

另外雖然還沒用到，但可以先建立資料庫

如果出現錯誤的話，很有可能是帳號密碼錯了，可以到config/dev.exs修改
